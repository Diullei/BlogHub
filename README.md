BlogHub
=======

## Installation

```$> git clone git://github.com/Diullei/BlogHub.git```

```$> cd .\BlogHub\engine```

```$> .\blobHub -i```

```$> .\blobHub -c```

```$> cd ..\```

```$> start_server```

Go to http:\\localhost:100

Enjoy!
